# leaf-resume

An online-tool helps to create better resume. [Live demo](https://littlesqx.github.io/leaf-resume/)

![](./static/0001.jpg)


## Build Setup

``` bash
git clone https://github.com/Littlesqx/leaf-resume.git

npm install

npm run dev # for dev

npm run build # for prod

```
